# 중력 (Gravity)

중력가속도 g를 측정하는 여러 가지 방법을 각각 독립된 인터랙티브 시뮬레이션으로 구현합니다.

## 확정된 시뮬레이션 (측정 방법별)

| # | 측정 방법 | 핵심 원리 | 폴더 | 상태 |
|---|---|---|---|---|
| 1 | 자유낙하 | h = ½gt² → g = 2h/t² | [free-fall/](./free-fall/) | ✅ 구현 완료 |
| 2 | 단진자 | T = 2π√(L/g) → g = 4π²L/T² | [pendulum/](./pendulum/) | PRD 초안 |
| 3 | 빗면(경사면) | a = g·sinθ → g = a/sinθ | [inclined-plane/](./inclined-plane/) | PRD 초안 |
| 4 | 용수철 진자 | mg = kx → g = kx/m | [spring/](./spring/) | PRD 초안 |

## 공통 사양 (5W1H 확정 사항)

- **Who**: 물리학I 수강 학생 (단진자 주기 공식은 유도 없이 제공형으로 사용)
- **When**: 교사 시연 + 학생 개별 탐구 + 수행평가 도구 겸용
- **Where**: 프로젝터(큰 글씨 모드) 및 학생 개인 기기(반응형) 모두 대응
- **How(스택)**: HTML/JS + Canvas 단일 파일 — 설치 없이 브라우저에서 실행

## 공통 필수 요소

- 매개변수 조절 UI (단위 표기 필수)
- 실시간 그래프/수치 표시
- 재생 / 일시정지 / 리셋
- 화면 내 물리 공식 설명 (펼침형)
- **측정 모드**: 학생이 시뮬레이션 안에서 직접 "측정"(스톱워치·자)하여 g를 계산하는 수행평가 모드
- **측정 오차 옵션**: 반응 시간 지연·눈금 해상도 등 현실적 오차 요인 ON/OFF
