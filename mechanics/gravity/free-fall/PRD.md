# PRD — 자유낙하로 g 측정 시뮬레이션

## 5W1H

| 항목 | 내용 |
|---|---|
| Why | 등가속도 운동 식 h = ½gt²를 "측정 도구"로 사용하는 경험. 수업 전개~정리 단계 |
| What | 공기저항 무시한 자유낙하. 낙하 높이 h와 낙하 시간 t를 측정해 g = 2h/t² 계산 |
| Who | 물리학I. 등가속도 운동 학습 직후 |
| When | 교사 시연 + 학생 탐구 + 수행평가 겸용 |
| Where | 프로젝터·개인 기기 모두 (반응형, 큰 글씨 모드) |
| How | 조절: 낙하 높이(m), 물체 종류(표시용). 피드백: 낙하 궤적, h-t 그래프, 스트로보 잔상 |

## 핵심 기능

1. 낙하 높이 슬라이더 (0.5 ~ 20 m)
2. **카운트다운(3·2·1) + 신호음 후 자동 낙하** — "낙하 시작" 클릭과 스톱워치 조작 사이의
   인위적 지연을 제거하고, 실제 실험처럼 순수 반응 시간만 오차로 남김
3. 시뮬레이션 내 스톱워치(Space 키) — 낙하/착지 신호음에 반응해 시작/정지
4. 자동 측정 모드 (광센서처럼 정확한 시간) — 오차 비교용
5. 측정값 기록 표 + 평균으로 g 계산, 참값 9.8 m/s²와 상대오차 표시
6. **실제 실험값 직접 입력** — 교실·계단에서 실측한 h, t를 입력하면 g를 계산해
   시뮬레이션 기록과 같은 표에서 비교 (실험 수업 보조 앱 역할)
7. h-t, v-t 실시간 그래프

## 학생 오개념 대비

- "무거운 물체가 빨리 떨어진다" → 질량이 달라도 낙하 시간이 같음을 나란히 낙하로 시각화
- "속도가 일정하게 떨어진다" → 스트로보 잔상 간격이 점점 벌어지는 것을 표시

## 완성 기준

- t = √(2h/g) 이론값과 시뮬레이션 낙하 시간 오차 < 0.1 %
- 스톱워치 수동 측정 시 반응 시간에 따른 오차가 g 계산값에 그대로 전파되는지 확인
