# PRD — 빗면(경사면)으로 g 측정 시뮬레이션

## 5W1H

| 항목 | 내용 |
|---|---|
| Why | 갈릴레이의 "시간 희석" 아이디어 체험 — 빗면으로 가속도를 줄여 측정 가능하게 함 |
| What | 마찰 무시(옵션으로 마찰 ON) 빗면 위 등가속도 운동. a = g·sinθ에서 g = a/sinθ 역산 |
| Who | 물리학I. 등가속도 운동 + 힘의 분해 학습 후 |
| When | 교사 시연 + 학생 탐구 + 수행평가 겸용 |
| Where | 프로젝터·개인 기기 모두 |
| How | 조절: 경사각 θ(°), 빗면 길이(m), 마찰계수(옵션). 피드백: 구간별 통과 시간, x-t·v-t 그래프 |

## 핵심 기능

1. 경사각 슬라이더 (5 ~ 60°) — 각도가 작을수록 시간 측정이 쉬워짐을 체험
2. 빗면 위 등간격 구간 센서(광센서 모방) — 구간 통과 시간 자동 기록
3. 기록 표 → x-t² 그래프 → 기울기(½a)에서 a 산출 → g = a/sinθ 계산
4. 수동 스톱워치 모드 (오차 체험) / 자동 센서 모드 비교
5. 마찰 ON 시 g가 실제보다 작게 측정되는 계통 오차 체험

## 학생 오개념 대비

- "빗면에서도 g로 가속된다" → 힘 분해 다이어그램으로 mg·sinθ만 운동 방향 성분임을 표시
- "기울기가 2배면 가속도도 2배" → sinθ는 비선형임을 그래프로 확인

## 완성 기준

- 마찰 0에서 시뮬레이션 가속도와 g·sinθ 오차 < 0.1 %
- x-t² 그래프가 직선으로 나타나고 기울기 역산 g가 참값과 일치
