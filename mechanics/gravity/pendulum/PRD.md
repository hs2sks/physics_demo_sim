# PRD — 단진자로 g 측정 시뮬레이션

## 5W1H

| 항목 | 내용 |
|---|---|
| Why | 주기 공식 T = 2π√(L/g)을 이용한 g 측정. 반복 측정과 평균의 중요성 학습 |
| What | 소각 근사(sinθ ≈ θ) 단진자. 공식은 물리학I 수준에 맞게 **유도 없이 제공** |
| Who | 물리학I (단진동 유도는 물리학II 범위이므로 공식 제공형) |
| When | 교사 시연 + 학생 탐구 + 수행평가 겸용 |
| Where | 프로젝터·개인 기기 모두 |
| How | 조절: 줄 길이 L(m), 초기 각도(°), 추 질량(kg). 피드백: 진자 운동, θ-t 그래프, 주기 자동/수동 측정 |

## 핵심 기능

1. 줄 길이 슬라이더 (0.1 ~ 3 m), 초기 각도 (1 ~ 30°)
2. "10회 왕복 시간 측정" 스톱워치 — 1회 측정보다 10회 평균이 정확함을 체험
3. 측정 기록 표 → T² vs L 그래프에 점 찍기 → 기울기(4π²/g)에서 g 산출
4. 질량 변화가 주기에 영향 없음을 즉시 확인 가능
5. 초기 각도가 크면(> 15°) 소각 근사가 깨져 오차가 커지는 것 표시

## 학생 오개념 대비

- "무거운 추가 빨리/느리게 흔들린다" → 질량 슬라이더를 바꿔도 주기 불변 시각화
- "진폭이 크면 주기가 길다(작다)" → 소각에서는 등시성, 대각에서는 근사 오차를 수치로 표시

## 완성 기준

- 소각(5°)에서 시뮬레이션 주기와 T = 2π√(L/g) 오차 < 0.5 %
- T²-L 그래프 기울기로 계산한 g가 9.8 ± 0.1 m/s² 범위
