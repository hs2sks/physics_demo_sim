# PRD — 용수철로 g 측정 시뮬레이션 (정적 평형법)

## 5W1H

| 항목 | 내용 |
|---|---|
| Why | 훅 법칙(F = kx)과 무게(W = mg)를 연결 — 힘 평형으로 g를 측정하는 정적 방법 |
| What | 용수철 상수 k를 아는 용수철에 질량 m을 매달아 평형 늘어남 x 측정 → g = kx/m |
| Who | 물리학I. 탄성력·힘의 평형 학습 후 (단진동 주기 공식은 물리학II라 정적법 우선) |
| When | 교사 시연 + 학생 탐구 + 수행평가 겸용 |
| Where | 프로젝터·개인 기기 모두 |
| How | 조절: 질량 m(g), 용수철 상수 k(N/m). 피드백: 늘어난 길이 눈금 확대경, x-m 그래프 |

## 핵심 기능

1. 질량 추 선택 (50/100/150/200 g) — 여러 질량으로 반복 측정
2. 눈금자 확대경 — 눈금 해상도(mm)에 따른 읽기 오차 체험
3. 기록 표 → x-m 그래프 → 기울기(g/k)에서 g 산출
4. 매단 직후 진동은 감쇠 후 평형 도달 (평형 상태에서 측정해야 함을 자연스럽게 학습)
5. (확장 옵션) 진동 주기 T = 2π√(m/k)는 g와 무관함을 보여주는 비교 탭 — 물리학II 연계

## 학생 오개념 대비

- "용수철이 늘어난 길이는 무게가 아니라 질량에 비례한다"는 혼동 → 달·화성 중력 프리셋으로 같은 질량이라도 늘어남이 달라짐을 시각화
- "진동 중 아무 때나 길이를 재면 된다" → 감쇠 진동 중 측정 시 오차 발생 체험

## 완성 기준

- 평형 늘어남 x = mg/k 이론값과 오차 < 0.1 %
- x-m 그래프 기울기로 역산한 g가 설정된 중력 프리셋 값과 일치 (지구/달/화성)
