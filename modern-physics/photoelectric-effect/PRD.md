# PRD — 광전효과로 플랑크 상수 h 측정 시뮬레이션

## 5W1H

| 항목 | 내용 |
|---|---|
| Why | 광전효과가 빛의 입자성(광자)의 증거임을 이해하고, 저지전압–진동수 관계에서 플랑크 상수 h를 "측정 도구"로 산출 |

## 기술 스택

HTML/JS + Canvas 단일 파일 (프로젝트 표준). 외부 파일·CDN 없음, 고정 크기 캔버스로 DPR 문제 원천 차단.

## 핵심 기능

1. **광원 설정** — 진동수 f 슬라이더(3.5~12 ×10¹⁴ Hz, 색으로 표시), 빛의 세기 슬라이더(0~100%)
2. **금속 설정** — 프리셋(Cs 2.14 / Na 2.36 / Zn 4.30 / Pt 5.64 eV) + 일함수 슬라이더, **미지 금속 X**(수행평가용, Φ 숨김)
3. **광전관 애니메이션** — 광음극에 부딪힌 광자가 조건(hf ≥ Φ) 충족 시 광전자 방출. 표면 전자(K_max, 청록)·심부 전자(에너지 손실, 주황) 구분 시각화
4. **실시간 계산값** — E = hf, Φ = hf₀, K_max = hf − Φ, V₀ = K_max/e
5. **두 모드(탭)**
   - 기본: 방출 여부·K_max 관찰 (세기 vs 진동수 대비)
   - 저지전압 측정: 외부 전압을 걸어 전류계로 광전류를 보며 **V₀를 직접 찾기**
6. **저지전압 측정 방식** — 수동(전압 슬라이더를 내려 전류가 0이 되는 순간을 학생이 판단해 기록 → 판단 오차 체험) vs 자동(정확한 V₀)
7. **측정 탐구(플랑크 상수)** — 여러 진동수에서 (f, V₀) 기록 → **V₀–f 그래프**에 점 표시 + 최소제곱 직선 적합 → 기울기 = h/e → **h 산출**, 참값 6.63×10⁻³⁴ J·s와 상대오차. x절편 = 한계진동수 f₀
8. **I–V 곡선** — 저지전압 측정 모드에서 전압에 따른 광전류 곡선 실시간 표시
9. **파동설 vs 입자설 비교** — 세기 증가/한계진동수 두 현상에 대한 고전 예측과 실제 결과 대비 카드
10. **CSV 저장**(엑셀 한글 호환 BOM) · **큰 글씨 모드** · **모바일 세로 레이아웃**

## 학생 오개념 대비

- "빛이 세면(밝으면) 전자의 에너지가 커진다" → 세기를 올려도 K_max 불변(전자 수만 증가)을 애니메이션·전류계로 시각화
- "빛을 오래/세게 쬐면 결국 전자가 나온다" → hf < Φ면 세기·시간 무관하게 방출 0
- "저지전압은 빛의 세기로 정해진다" → V₀는 진동수에만 의존, 세기와 무관함을 측정으로 확인
- "광자 여러 개가 힘을 합쳐 전자 하나를 때린다" → 광자-전자 1:1 상호작용 강조

## 완성 기준

- E = hf, K_max = hf − Φ, V₀ = K_max/e 수치가 이론값과 일치 (콘솔 검증)
- 자동 측정으로 (f, V₀)를 3점 이상 기록해 직선 적합 시 산출된 h의 오차 < 1 %
- 세기만 바꿀 때 K_max·V₀ 불변, 전류(포화값)만 변함
- hf < Φ에서 전압·세기 무관하게 광전류 = 0
