시간 관계 힌트
• t₁ > t₀ : 빛이 달아나는 거울을 향해 이동 → 더 오래 걸림
• t₂ < t₀ : 거울이 반사된 빛 쪽으로 이동 → 빨리 돌아옴
• t₃ = t₄ = γt₀ : 수직 방향은 대칭, 시간 팽창만 발생
• t₁ + t₂ = 2γt₀ ,   t₁ · t₂ = t₀² → 두 시간의 기하평균 = t₀
