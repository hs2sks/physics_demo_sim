두 파동의 보강간섭과 상쇄간섭 가상 실험 시뮬레이션

중첩의 원리 (Superposition)

두 파동이 같은 공간을 지날 때, 합성 변위는 각 파동 변위의 합과 같습니다.
y = y₁ + y₂

보강간섭 (Constructive)

두 파동의 위상차가 0° (같은 위상)일 때 진폭이 최대로 커집니다.
경로차 = nλ (n = 0, 1, 2, …)

상쇄간섭 (Destructive)

두 파동의 위상차가 180° (반대 위상)일 때 진폭이 최소로 작아집니다.
경로차 = (n + ½)λ
