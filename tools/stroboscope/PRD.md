# PRD — 스트로보스코프 (동영상 다중섬광 분석기)

## 5W1H

| 항목 | 내용 |
|---|---|
| Why | 빠른 운동(자유낙하·단진자 등)을 일정 시간 간격 사진으로 "정지"시켜 분석하는 스트로보스코프 원리 체험 + 실제 실험 영상의 정량 분석 |
| What | 학생이 스마트폰으로 찍은 영상 → 일정 간격 프레임 추출 → 다중섬광사진 합성 → 위치 클릭 → x-t, v-t 데이터/그래프/CSV |
| Who | 물리학I. 등가속도 운동·단진동 실험 분석용 |
| When | 실험 후 분석 단계 (개별/모둠), 교사 시연 |
| Where | GitHub Pages 배포 — 학생이 링크로 접속, 스마트폰/PC 모두 |
| How | 4단계 워크플로: ① 영상 불러오기 ② 다중섬광사진 ③ 위치 찍기(보정 포함) ④ 결과 분석 |

## 핵심 기능

1. **영상 업로드** — 파일 선택, 분석 구간(시작/끝)과 섬광 간격 Δt 설정,
   슬로모션 영상 배율 보정(240fps → 8배 등)
2. **다중섬광사진 합성** — 어둡게/밝게/배경차이 3가지 합성 모드, PNG 저장
3. **기준 거리 보정** — 화면에서 두 점 클릭 + 실제 길이 입력 → m/px 환산
4. **프레임별 위치 찍기** — 프레임을 넘기며 물체 위치 클릭 (Tracker 방식)
5. **정량 분석** — x-t, y-t, v-t 그래프, 등가속도(포물선) 피팅 → 가속도 a 산출
   (자유낙하 영상이면 a ≈ g), 진동 주기 자동 추정(단진자용), CSV 다운로드
6. 스마트폰 세로 화면 대응

## 학생 오개념/실수 대비

- 보정 없이 픽셀 좌표로 계산 → 보정 전에는 그래프에 "픽셀 단위" 경고 표시
- 슬로모션 영상을 실시간으로 착각 → 배율 입력칸과 설명 제공
- 화면 좌표는 아래가 + 방향 → 축 방향을 명시 (낙하 분석에 편리)

## 완성 기준

- 알려진 가속도로 만든 합성 영상 분석 시 피팅 가속도 오차 < 5%
- 프레임 추출·합성·클릭·그래프·CSV 전 과정이 파일(오프라인)과 Pages(온라인) 모두에서 동작

## 한계 (화면에 명시)

- 시간 정밀도는 영상 프레임레이트에 제한됨 (30fps → 1/30초).
  빠른 운동은 스마트폰 슬로모션(120/240fps) 촬영 권장
